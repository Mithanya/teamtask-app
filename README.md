# TeamTask – Collaborative Task Manager

TeamTask is a responsive web-based task management application designed to help teams organize, assign, and track tasks efficiently. It provides a clean user interface with essential productivity features and analytics.

Live Demo:
https://mithanya.github.io/teamtask-app/

Repository:
https://github.com/Mithanya/teamtask-app

Features:
- User authentication (Login and Signup using LocalStorage)
- Role-based access (Manager and Member)
- Create, edit, delete, and complete tasks
- Assign tasks to team members
- Task filtering by assignee and status
- Search functionality for tasks
- Due date tracking with overdue and reminder notifications
- Drag and drop task reordering (SortableJS)
- Analytics dashboard with charts (Chart.js)
- Dark mode support
- Responsive design for mobile and desktop

Technologies Used:
- HTML5
- CSS3
- JavaScript (Vanilla)
- Chart.js
- SortableJS
- LocalStorage

How to Run Locally:
1. Clone the repository:
   git clone https://github.com/Mithanya/teamtask-app.git

2. Navigate to the project folder:
   cd teamtask-app

3. Open index.html in your browser
